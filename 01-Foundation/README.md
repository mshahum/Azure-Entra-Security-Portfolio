# Foundation Labs

This phase contains the foundational Microsoft Entra ID and Azure security labs in this portfolio.

These labs focus on the core identity and access controls that organizations commonly use to protect users, govern privileged access, review stale permissions, and enable secure account recovery.

## Foundation Projects

1. **Conditional Access + MFA**  
   Protects privileged accounts and helps reduce risky sign-in scenarios.

2. **Privileged Identity Management (PIM)**  
   Demonstrates just-in-time privileged access for Microsoft Entra roles and Azure resources.

3. **Access Reviews**  
   Shows how organizations review and validate whether sensitive access is still required.

4. **Authentication Methods + SSPR + Registration Campaign**  
   Demonstrates secure user registration, authentication method governance, and self-service password recovery.

## Purpose of This Phase

The purpose of the Foundation phase is to build a strong practical base in:

- identity protection
- privileged access control
- access governance
- authentication and recovery security

This phase is designed to support progression toward more advanced Microsoft security and Azure security engineering labs in later portfolio stages.
